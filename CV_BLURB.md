## Medium Version

Developed a browser-based AR-style ultrasound simulator that displays a synthetic 2D ultrasound slice in the correct 3D position and orientation relative to a movable virtual probe. The project uses React, TypeScript, Three.js, and procedural image generation to demonstrate how in-situ visualization can reduce the spatial ambiguity of conventional ultrasound views.

## Long Version

Built a browser-based AR-style ultrasound visualization simulator that explores how in-situ slice display can support 3D interpretation in image-guided procedures. The prototype connects a movable virtual probe, a physically attached imaging plane, simplified synthetic anatomy, and procedurally generated ultrasound-like rendering so the displayed 2D slice remains spatially registered to the anatomy it represents. The project is relevant to AR ultrasound guidance, human-computer interaction, and medical visualization because it demonstrates how interface design and geometric registration can reduce the cognitive burden of interpreting conventional detached ultrasound monitor views.

## GitHub Description

AR-style ultrasound visualization simulator built with React, TypeScript, Three.js, and procedural rendering. A virtual probe drives a synthetic 2D ultrasound slice displayed in its correct 3D position and orientation inside a simplified torso phantom.
